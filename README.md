# 🧁 Doceria Delicatto

Bem-vindo ao repositório da **Doceria Delicatto**! Este é um projeto de website moderno e responsivo para uma confeitaria especializada em doces artesanais.

## 🌐 Link do Projeto

**Deploy:** [https://doceria-delicatto.vercel.app](https://doceria-delicatto.vercel.app)

---

## 📋 Sobre o Projeto

A **Doceria Delicatto** é um empreendimento dedicado à criação de doces artesanais de alta qualidade. Este website foi desenvolvido para apresentar os produtos, facilitar o contato com clientes e proporcionar uma experiência visual agradável e intuitiva.

### 👤 Responsável pelo Atendimento

**Leticia** é a responsável pelo atendimento da Doceria Delicatto e **atende exclusivamente via WhatsApp**. Para fazer pedidos ou obter mais informações sobre os produtos, entre em contato através do WhatsApp.

## 📁 Estrutura do Projeto

```
Doceria_Delicatto/
├── index.html          # Arquivo principal HTML com toda a estrutura do site
├── style.css           # Estilos CSS com design responsivo
├── script.js           # Lógica JavaScript para interatividade
├── images/             # Pasta contendo imagens dos produtos
└── README.md           # Este arquivo
```

### Descrição dos Arquivos

#### **index.html** (22.9 KB)
- Contém a estrutura completa do website
- Inclui seções de apresentação, galeria de produtos e contato
- Código semântico e bem organizado

#### **style.css** (25.7 KB)
- Define toda a estilização visual do site
- Implementa design responsivo com media queries
- Animações e transições suaves
- Paleta de cores profissional

#### **script.js** (12.8 KB)
- Funcionalidades interativas do website
- Manipulação do DOM
- Validações e eventos do usuário
- Integração com dados de produtos

#### **images/** 
- Pasta destinada ao armazenamento de imagens dos produtos
- Fotos de doces e confeitarias

## 🚀 Desenvolvimento

### Ferramentas Utilizadas
- Editor de Código: VS Code / Sublime / Outro
- Versionamento: Git & GitHub
- Deploy: Vercel

### Como Executar Localmente

1. Clone o repositório:
```bash
git clone https://github.com/EduardoSato22/Doceria_Delicatto.git
```

2. Navegue até a pasta do projeto:
```bash
cd Doceria_Delicatto
```

3. Abra o arquivo `index.html` em seu navegador ou use um servidor local:
```bash
python -m http.server 8000
# ou
npx http-server
```

4. Acesse `http://localhost:8000` em seu navegador
